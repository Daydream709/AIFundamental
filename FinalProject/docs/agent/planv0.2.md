# 数据科学与工程学院 Project Two 时序预测完整攻坚计划（顶配精简版）

---

## 一、任务选型

多模态长周期时间序列预测，同时做全监督预测 + Zero-Shot 基础模型预测双向对比。

**输入**：时序数值 + 外源文本新闻/政策 + 时序转图像模态  
**输出**：未来长序列预测 $H\times C \to F\times C$

## 三、模型阵容（7个，精简后）

### 基线模型

- **DLinear**：极简MLP强基线
- **PatchTST**：经典分块Transformer（ICLR 2023）
- **iTransformer**：长期预测传统SOTA（AAAI 2024 Best Paper）

### 2024-2025 SOTA

- **TimeMixer++**：ICLR 2025通用时序新SOTA
- **SST / S2TX**：Mamba+Transformer混合架构，线性复杂度
- **TimeKAN / AdaKAN**：KAN网络用于时序，可解释性强

### 时序基础大模型（Zero-Shot）

- **Chronos2**：亚马逊时序基础大模型

> 覆盖架构：MLP / Transformer / Mamba / KAN / 基础模型

## 四、核心创新点（5个）

### 创新1：KAN-Enhanced时序预测网络

将iTransformer部分MLP前馈层替换为KAN层，引入自适应频率分解（趋势+季节+残差）。

### 创新2：Mamba-Transformer双专家路由架构

- 长程专家：Mamba/SSM处理低频全局趋势（线性复杂度）
- 短程专家：局部窗口Transformer处理高频局部波动
- 动态路由器：根据输入序列频域特征自适应分配权重

### 创新3：跨模态对比对齐融合

时序数值 + 外源文本(LLM Embedding) + 时序灰度图(VisionTS风格)。设计对比损失函数让同一时刻不同模态表示靠近，再经自适应门控网络动态融合。

### 创新4：共形预测区间估计

分位数回归 + 共形预测（Conformal Prediction），输出具有理论保证的95%置信区间。

### 创新5：模型仲裁集成系统

设计轻量级元学习仲裁器，根据输入序列统计特征（熵、趋势强度、周期性）动态选择最优模型或加权集成。
