# 二、核心创新点（专属加分，区别普通同学）

## 创新1：多模态自适应融合模块

原生只用时序/文本/图像，我设计**三模态注意力对齐融合**：
时序数值 + 外源文本LLM语义特征 + 时序灰度图视觉特征
设计跨模态门控融合机制，解决模态异构、特征分布不一致问题。

## 创新2：改进iTransformer注意力机制

1. 引入**频域滤波**替代部分原始注意力，降低计算复杂度
2. 加入**周期分解+非平稳性自适应归一化**
3. 实现轻量化改进：参数量下降、精度不下降

## 创新3：概率预测损失函数改造

不只用MSE，改用**高斯分布概率损失**，输出预测均值+置信区间，做不确定性估计，远超普通点对点预测。

## 创新4：模型横向四大维度系统对比

- 预测精度（MAE/MSE）
- 模型参数量大小
- 推理耗时
- 全监督 vs Zero-Shot 泛化能力

## 创新5：搭建简易可视化展示页面

做网页端可视化：上传时序序列 → 选择模型 → 实时展示各模型预测曲线对比。
