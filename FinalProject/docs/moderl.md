DLinear 优点
速度爆炸快：时间复杂度 O (H)，Transformer 是 O (H²)，长序列差距巨大；
参数量极少，显卡要求低，普通 CPU 就能跑；
超长预测效果稳定：历史窗口 H 越大，预测越准；很多 Transformer 长窗口效果反而变差；
结构简单，调参极少，不容易过拟合；
可解释性强，线性权重能直观看出历史如何影响未来。
DLinear 缺点
只有纯线性拟合，完全无法捕捉极度复杂、非线性突变规律（比如突发故障、极端异常值）；
没有非线性层，对于高度非线性、无明显趋势周期的数据效果不如 Autoformer/Mamba。



PatchTST 优缺点
优点
长时序预测效果强，远超传统逐点 Transformer；
Patch 降 Token 数量，计算、显存开销大幅降低；
通道独立设计适配多变量场景，抗干扰；
支持对比学习自监督预训练（InfoNCE 损失），少量数据也能训好；
结构标准，直接复用你学过的 Transformer 三件套。
缺点
相比 DLinear 还是更重、训练更慢；
超参数敏感（Patch 长度 P、步长 S 调参影响巨大）；
极短历史窗口（H 很小）场景下优势不明显。



TimesNet 优缺点
优点
显式建模双重周期规律，同比、环比信息同时捕捉，拟合周期性数据能力极强；
FFT 自动找周期，不用人工提前设定窗口；
2D 卷积代替自注意力，参数量更少、训练更快；
模块化 TimesBlock 可随意堆叠，残差保证深层可训练；
通用模型，一套架构适配 5 大类时序任务。
缺点
极度无规律、无明显周期的噪声数据（纯随机波动），2D 折叠没有增益；
FFT 频域计算有少量额外开销，比极简 DLinear 慢；
超参数 K（提取周期数量）需要调参，K 太大容易引入无用周期。


mamba 优缺点
优点：线性复杂度，超长序列速度、显存碾压 Transformer；卷积 + 选择性 SSM 同时抓局部波动与长周期全局依赖，通用轻量化；
缺点：单向扫描有顺序偏见，固定隐状态压缩历史存在信息损耗；短序列精度不及注意力，超参数敏感、可解释性差。

先做四类架构对应绑定：
1. DLinear → **纯线性分解架构（单层线性回归）**
2. PatchTST → **基于Patch的编码器式Transformer架构**
3. TimesNet → **频域变换+2D卷积时序卷积架构**
4. Mamba → **选择性状态空间SSM线性时序架构**

# 极简横向对比速记
1. Linear：最轻最快，只能拟合简单线性规律，表达能力最弱；
2. CNN：擅长局部+周期，速度中等，远距离依赖建模能力有限；
3. Transformer：全局建模能力天花板，但长序列计算、显存成本极高；
4. SSM：线性速度+接近Transformer的长依赖能力，超长时序最优，但可解释差、单向有局限。

# 四款模型完整分步训练流程（适中详细，循序渐进）
统一前置通用步骤：
数据划分训练/验证/测试集；输入统一用RevIN可逆实例归一化；优化器统一AdamW；早停+weight decay防过拟合；可选高斯概率输出+共形预测做区间。

## 一、DLinear（纯线性分解模型）
1. 数据预处理：原始多变量时序送入RevIN，按每条样本自身均值方差归一化，消除量级差异。
2. 双通道线性映射：设置趋势、季节两条独立单层线性支路，无激活、无隐藏层；每条变量单独运算，变量之间无交互。
3. 特征融合：趋势分支输出 + 季节分支输出，得到归一化后的初步预测。
4. 反归一化：使用该样本归一化时记录的均值方差还原真实数值。
5. 损失计算与参数更新：用MSE或高斯NLL计算误差；反向传播更新两路线性层权重；验证集监控loss，触发早停停止训练。
6. 推理增强（可选）：高斯输出得到原生置信区间，再用验证集完成共形校准，输出带保障的预测上下限。

## 二、PatchTST（通道独立Patch Transformer）
1. 归一化：输入时序经过RevIN实例归一化。
2. 分变量切块：拆分所有变量，每个变量单独处理；将单变量长时序切分成连续Patch时间块，块为基础Token。
3. Patch嵌入：每个Patch通过线性层映射为固定维度向量，搭配位置编码区分时间先后。
4. 多层Encoder堆叠：每层包含多头自注意力+FFN残差；注意力仅在**同一个变量的Patch之间**计算，完全无跨变量交互；FFN引入非线性拟合局部波动。
5. 预测输出头：Encoder输出特征接线性层，输出未来F步预测（均值+方差双分支做概率预测）。
6. 反归一化+训练更新：RevIN还原真实数值；高斯NLL损失反向更新嵌入、注意力、输出层全部参数；验证集早停防止过拟合。
7. 可选校准：验证集校准共形预测，推理输出可靠不确定性区间。

## 三、TimesNet（频域2D卷积架构）
1. 归一化：原始时序执行RevIN实例归一化。
2. FFT频域周期提取：对单变量时序做傅里叶变换，筛选多条显著周期分量。
3. 时序2D折叠：按照提取到的周期长度，把一维时间序列重排成二维矩阵，一行代表一个完整周期。
4. 多尺度卷积提取特征：使用Inception多尺度2D卷积，同时捕捉周期内短期波动、跨周期同比规律；卷积搭配GELU非线性，多层残差堆叠加深特征提取能力。
5. 特征融合+预测头：融合所有周期卷积特征，通过线性层映射得到未来时序预测。
6. 反归一化与训练：RevIN还原真实业务数值；MSE/Gaussian NLL反向更新FFT映射、卷积、输出层参数；验证集早停终止训练。
7. 可选：叠加概率输出、共形预测得到校准区间。

## 四、Mamba（选择性SSM状态空间架构）
1. 归一化：输入多变量时序经过RevIN实例归一化。
2. 局部特征提取：因果一维卷积抓取相邻时间步短期局部波动。
3. 选择性SSM单向扫描：卷积特征送入SSM模块，通过Δₜ/Bₜ/Cₜ三组动态门控控制历史信息写入、遗忘、读取；固定尺寸隐状态存储全局超长时序信息，线性计算复杂度。
4. 多层MambaBlock堆叠：卷积+SSM组合块搭配残差连接多层叠加，逐步提取长短时序依赖；通道独立设计，变量分开建模无原生跨变量交互。
5. 预测输出：顶层特征接线性头，输出预测均值与方差。
6. 反归一化与训练：RevIN还原真实数值；高斯NLL/MSE反向更新卷积、SSM门控、输出层权重；依靠早停、weight decay抑制过拟合。
7. 推理拓展：高斯生成原生区间，共形预测校准后输出带统计保证的预测范围。
