f-string 自由化

PEP 701

解除了 f-string 的诸多语法限制，现在支持嵌套任意引号、反斜杠转义和多行表达式，无需再为引号冲突烦恼。 

# Python 3.12 之前：引号冲突会导致语法错误
# 现在可以嵌套相同类型的引号
name = "Python"
version = 3.12

# 1. 支持嵌套引号（之前必须用不同引号）
s = f"{'Hello' + ' ' + name}!"

# 2. 支持反斜杠（之前完全禁止）
path = f"C:\\Users\\{name}\\file.txt"
regex = f"Pattern: \\d+"

# 3. 多行表达式（无需反斜杠续行）
result = f"""
计算结果: {
    100 + 200 + 
    300 + 400
}
"""
print(s)
print(path)

泛型语法简化

PEP 695

全新的类型参数声明语法，使用方括号直接声明类型变量，告别繁琐的 TypeVar 定义，让泛型编程更直观。 

# Python 3.12 新语法：直接声明类型参数 [T]
# 无需再写：from typing import TypeVar; T = TypeVar('T')

def process[T](data: T) -> T:
    """返回输入数据的泛型函数"""
    return data

class Container[T]:
    """泛型容器类，T 在类定义时声明"""
    def __init__(self, value: T) -> None:
        self.value = value
    
    def get(self) -> T:
        return self.value

# 类型推断示例
num_box = Container(42)      # 自动推断为 Container[int]
str_box = Container("hello") # 自动推断为 Container[str]

# 函数使用
print(process(100))          # 输出: 100
print(num_box.get())         # 输出: 42
